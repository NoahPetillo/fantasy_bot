How to find your ESPN cookies (60 seconds)
  1. Open ESPN Fantasy in a new tab and make sure you're logged in.
  2. Chrome / Edge: press F12 (or ⌥⌘I on Mac) → Application tab → Cookies → https://www.espn.com.
    Safari: enable the Develop menu (Settings → Advanced), then Develop → Show Web Inspector → Storage → Cookies.
    Firefox: F12 → Storage tab → Cookies.
  3. Find the row named espn_s2 and copy its whole value (it's long, ~200+ characters).
  4. Find the row named SWID and copy its value including the curly braces, like {1234ABCD-…}.
  5. Paste both below. Cookies last months; if they expire, repeat these steps.
